Condicionais e Objetos

Declaração de Objetos em JavaScript

O objeto é um tipo de dado não primitivo, composto por propriedades e valores. A referência do objeto não pode ser alterada, mas as propriedades podem ser modificadas ao longo do código.

Para declarar objetos em JavaScript, pode-se utilizar let, const ou var. Em seguida, cria-se uma variável, utiliza-se o sinal de igual (=) e abre-se chaves { }. Dentro das chaves, são criadas as características desejadas, associando valores a cada uma delas.

Exemplo: 

        let aluno = {
            nome: "Karla", 
            nota: 7, 
            matricula: 234
        }

Estruturas Condicionais

Condicional Simples

As estruturas condicionais servem para alterar o fluxo de execução de um algoritmo. Após o teste de condição, se a condição for verdadeira, o bloco interno é executado. Caso contrário, o fluxo do código segue após a estrutura condicional.

Exemplo em pseudocódigo: 

        if (condição == verdadeira) {
            comandos executados se for verdadeiro
        } else {
            comandos executados se for falso
        }
